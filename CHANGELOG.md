# Change Log

All notable changes to the "vscode-time-analytics" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [Unreleased]

- **Multi-user Support:** Future support for tracking time per user in shared workspaces (authenticated vs anonymous).
- **Git Integration:** Syncing stats via Git for team analytics.

## [0.1.0] - 2025-12-14

### Added

- Initial release of Time Analytics.
- Real-time tracking of Active vs Idle time.
- Global Stats dashboard aggregating all workspaces.
- Recursive file tree view with folder aggregation.
- "Deleted Files" bin to preserve history of deleted items.
- Local JSON storage for privacy.
